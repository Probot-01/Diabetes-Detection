Sample audio files are not included in this repository.

To test the system:
1. Record yourself saying 'aaah' steadily for 5 seconds
2. Save as a .wav file
3. Upload to the app interface

Recording requirements:
- Quiet room with minimal background noise
- 5 seconds sustained vowel sound
- Standard microphone or smartphone mic
- Save as .wav format (convert from other formats if needed)
